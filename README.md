# Alejandro Arteaga - Personal Portfolio Website

A static, single-page personal portfolio website built with Astro and hosted on GitHub Pages
at [ajarteag.github.io](https://ajarteag.github.io). Terminal-themed, minimal, and responsive.

## Project Roadmap

- [ ] **Phase 1 — Scaffold:** Initialize the Astro project and wire `npm run build` → `dist/`
      so the existing GitHub Pages deploy workflow publishes the site.
- [ ] **Phase 2 — Content & layout:** Build the single-page scroll with these sections:
  - **About** — hero intro (name, tagline, GitHub + LinkedIn links, resume download) merged
    with a short bio.
  - **Education** — USC (MS, Intelligent Robotics) and Carnegie Mellon (BS, Physics/Astrophysics).
  - **Experience** — NVIDIA, Qualcomm, CMU TartanAUV, National Instruments, CMU Physics.
  - **Projects** — DLIO edge-computing optimization and the multimodal GenAI nutrition assistant.
- [ ] **Phase 3 — Styling:** Apply the terminal/developer aesthetic, responsive layout, and
      resume download button.
- [ ] **Phase 4 — Polish & ship:** Accessibility pass, meta/SEO tags, and verify the GitHub
      Pages deployment.
